# Environment Setup

## Purpose

The first phase of this project focused on preparing a stable Linux environment to host the osTicket help desk application. Establishing a properly configured server provides the foundation for all future services, including the web server, database, and application components.

## Environment

| Component               | Configuration                     |
| ----------------------- | --------------------------------- |
| Operating System        | Ubuntu Server 22.04 LTS           |
| Virtualization Platform | Oracle VM VirtualBox              |
| Project                 | osTicket Enterprise Help Desk Lab |

## Objectives

During this phase, the following objectives were completed:

* Installed Ubuntu Server 22.04 LTS
* Configured the virtual machine
* Verified network connectivity
* Updated the operating system
* Prepared the server for application deployment

## Initial System Preparation

After installation, the operating system was updated to ensure the latest security patches and package versions were installed.

Common administrative tasks performed during this phase included:

* Updating package repositories
* Upgrading installed packages
* Verifying network connectivity
* Confirming system resources
* Preparing the server for software installation

## Verification

Before moving to the next phase, the following items were confirmed:

* Ubuntu Server booted successfully
* Internet connectivity was available
* Administrative access was functional
* Package repositories were reachable
* System updates completed successfully

## Outcome

The server was successfully prepared for deployment of the LAMP stack. With the operating system configured and verified, the environment was ready for installing Apache, PHP, MariaDB, and the osTicket application.
